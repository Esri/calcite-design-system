calcite-navigation

Calcite Nav is a positioning and layout component that supports consistent application and product igationigation. It contains slots that position calcite-navigation-logo and calcite-navigation-logo. It has slots for positioning calcite-menu as well as other interactive content. The primary level is larger and more emphasized as it should be the first item populated and consistent across products. The secondary and tertiary level is slightly shorter to provide visual hierarchy.
